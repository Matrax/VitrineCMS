Description détaillée

Classe abstraite représentant un conteneur HTML.

Classe abstraite représentant un element HTML. Un element HTML possède un code html et un identifiant. L'identifiant va permettre aux elements de modfier leurs contenu à l'aide de la page administrateur. @type html-content
